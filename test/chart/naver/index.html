장점

  • 심플하다
  • 접근이 용이하다
  • 브라우저에 독립적이다
  • 웹접근성이 고려돼 있다


단점

  • 사이즈 조절 api 미지원
  • 색상을 이미지패턴으로 표시하는 기능 미지원(단, 색상, 굵기, 투명도, 폰트 등은 옵션과 css로 수정가능)
  • 커스터마이징이 상당히 제한적



* 네이버차트는 차트로서의 기능보다는 단순히 브라우저에 따라 vml, svg를 분기해서 똑같은 모양으로 그려주는 것에 중점을 둔듯한 느낌.
