장점

  • 강력한 그래픽관련 API 제공
  • 리사이징을 위한 API가 따로 있어서 리사이징이 쉽다(아랫 샘플참조)
  • 브라우저에 독립적이다
  • 웹접근성이 고려됨(페이지에 데이타정보가 담긴 테이블(표)만 있으면 이를 바탕으로 그려준다.)


단점

  • 차트 전용 라이브러리가 아니다.
  • 라파엘사이트에서 제공되는 차트의 기능과 종류는 상당히 제한적이다.
  • 자체적으로 라파엘을 이용해서 차트를 만들어야 한다.(하지만 전세계 구루들에 의해 만들어진 차트들이 많이 공유되어 있기는 합니다.)
